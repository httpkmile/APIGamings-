# Assets Directory

This directory contains static assets used by the Gamings API application.

## Files

- `header.jpg` - Main header image displayed at the top of the application

## Usage

Assets are referenced in the application using relative paths:

```python
st.image("assets/header.jpg", width=None, use_container_width=True)
```

## Guidelines

- Keep images optimized for web use
- Use appropriate file formats (JPG for photos, PNG for graphics)
- Maintain responsive design principles
